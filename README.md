# Phonexa-MVC
Phonexa-MVC is a PHP-based MVC web application for managing products and brands with full CRUD functionality and MySQL database integration. The application includes a secure admin authentication system with login and logout functionality, session management, and password hashing for enhanced security. It allows administrators to manage their profile information and securely access protected admin routes. The system enables administrators to efficiently manage products and brands through an intuitive admin dashboard while maintaining a structured MVC architecture for better code organization and scalability.
